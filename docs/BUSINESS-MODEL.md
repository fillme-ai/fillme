# Business Model: OpenWhip

**Date**: 2026-04-15

---

## Lean Canvas

### 1. Problem

| 순위 | 문제 |
|------|------|
| 1 | 기업 자체 채용 사이트(삼성, 현대, 네이버 등)마다 동일 정보를 반복 입력해야 함. 기업당 30~60분 소요 |
| 2 | 기업 채용 사이트에는 "이력서 가져오기" 기능이 없고, 양식이 전부 다름 |
| 3 | 공채 시즌에 10~20개 기업 동시 지원 시 며칠을 입력에만 소모 |

### 2. Customer Segments

**Primary (B2C)**:
- 공채 준비 신입 구직자 (20대) — 연 50만명+
- 이직 준비 경력직 (30대) — 연 80만명+

**Secondary (B2B)**:
- 대학교 취업지원센터 (~200개교)
- 헤드헌팅사

### 3. Unique Value Proposition

> **"이력서 한 번 올리면, 어떤 기업 채용 사이트든 자동 입력"**

- 사람인/잡코리아가 커버하지 못하는 기업 자체 채용 사이트에 특화
- 범용 AI 폼 매퍼로 사이트별 하드코딩 없이 대응

### 4. Solution

| 구성요소 | 기능 |
|----------|------|
| AI 이력서 파서 | PDF/이미지 → 구조화 데이터 (한국어 특화) |
| 범용 AI 폼 매퍼 | 어떤 채용 사이트든 폼 label을 분석하여 자동 매핑 |
| 크롬 익스텐션 | 채용 사이트 감지 → 원클릭 자동 입력 |
| 마스터 이력서 대시보드 | 웹 기반 이력서 중앙 관리 |

### 5. Channels

| 채널 | 전략 |
|------|------|
| 구직 커뮤니티 (블라인드, 에브리타임) | 공채 시즌 바이럴 |
| 숏폼 콘텐츠 (유튜브/틱톡/릴스) | "10개 기업 30분 만에 지원" 타임랩스 |
| Chrome 웹스토어 | SEO: "입사지원 자동입력" |
| 대학교 취업특강/박람회 | 신입 구직자 대량 확보 |
| 레퍼럴 | 친구 초대 시 무료 횟수 추가 |

### 6. Revenue Streams

**B2C**:
| 플랜 | 가격 |
|------|------|
| Free | 0원 (파싱 월 3회, 자동입력 월 5회) |
| Basic | 월 4,900원 (무제한 파싱, 자동입력 월 30회) |
| 시즌패스 | 19,900원/3개월 (무제한 — 공채 시즌 최적) |
| 건당 | 1,500원/회 |

**B2B**:
| 고객 | 모델 |
|------|------|
| 대학교 취업지원센터 | 연 500~2,000만원 |
| 헤드헌팅사 | 인당 월 19,900원 |

### 7. Cost Structure

| 항목 | 초기 (월) | 성장기 (월) |
|------|----------|------------|
| 인건비 | 2,500만원 | 5,000만원 |
| AI/LLM API | 200만원 | 1,000만원+ |
| 인프라 | 100만원 | 500만원 |
| 마케팅 | 300만원 | 1,500만원 |

### 8. Key Metrics

| 지표 | 12개월 목표 |
|------|------------|
| 크롬 익스텐션 설치 수 | 50,000+ |
| MAU | 20,000+ |
| Free → Paid 전환율 | 5~8% |
| MRR | 3,000만원+ |
| 파싱 정확도 | 95%+ |

### 9. Unfair Advantage

- 기업 자체 채용 사이트 타겟 → 사람인/잡코리아가 복제 불가 (구조적 이점)
- 범용 AI 폼 매퍼 기술 + 한국어 이력서 파싱 데이터 축적
- 사용자 마스터 이력서 + 지원 이력 → 전환 비용

## 수익화 타임라인

```
Phase 1 (0~6개월): 유저 확보
├── 전체 기능 무료 (사용량 제한 없음)
├── 목표: 익스텐션 10,000 설치
└── 수익: 0원

Phase 2 (6~12개월): 소프트 페이월
├── Free 플랜 제한 도입
├── 대학교 B2B 파일럿 3~5개교
└── 목표: MRR 1,000만원

Phase 3 (12~24개월): B2B 확장
├── 헤드헌팅사/대학교 엔터프라이즈
├── 유료 전환율 5~8%
└── 목표: MRR 5,000만원+

Phase 4 (24개월~): 플랫폼화
├── 지원 관리 대시보드, JD 매칭
├── 해외 확장 (일본, 동남아)
└── 목표: ARR 100억원+
```

## 경쟁 분석

| 서비스 | 차이점 |
|--------|--------|
| 사람인/잡코리아 자체 기능 | 자사 플랫폼 내에서만 작동. 기업 자체 채용 사이트 커버 불가 |
| Chrome 자동완성 | 단순 텍스트만 가능. 드롭다운/날짜/복합 양식 불가 |
| Simplify (미국) | 한국 기업 채용 사이트 미지원. 한국어 파싱 불가 |
| ChatGPT/Claude | 이력서 작성은 도와주지만 폼 자동 입력 불가 |

**핵심**: 한국 기업 자체 채용 사이트에 범용 대응하는 AI 자동 입력 도구는 현재 **존재하지 않음**
