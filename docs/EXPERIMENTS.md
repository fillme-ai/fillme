## 실험 설계

### 실험 A: 랜딩페이지 (Fake Door) — 즉시 시작

- **검증**: H1 + H5
- **방법**: 1페이지 랜딩 제작. "이력서 한 번 올리면, 삼성/현대/네이버 채용 사이트 자동 입력" 카피 + 목업 GIF + 이메일 폼. 블라인드/에브리타임/오픈카톡에 배포
- **성공 기준**: 전환율 >= 40%, CTR >= 5%
- **노력**: 1인 1일
- **기간**: 제작 1일 + 수집 7일 = 8일
- **비용**: 0~3만원

### 실험 B: 가격 테스트 (Painted Door) — A 직후

- **검증**: H2
- **방법**: 랜딩 하단에 요금제 3가지 (무료/월 4,900원/시즌패스 19,900원). 클릭 시 "출시 준비 중" 안내. A/B 테스트
- **성공 기준**: 유료 플랜 클릭률 >= 25%
- **노력**: 0.5인일
- **기간**: 7일
- **비용**: 0원

### 실험 C: 컨시어지 MVP (Wizard of Oz) — A/B 통과 후

- **검증**: H3 + H4
- **방법**: 15명 모집 → 이력서를 구글폼으로 받음 → **수동으로** 파싱/정리 → 원격 화면공유로 기업 채용 사이트에 대신 입력. "AI 처리 중"으로 안내
- **성공 기준**: 자동 입력 요청률 >= 60%, 7일 내 재요청률 >= 50%
- **노력**: 3인일
- **기간**: 12일
- **비용**: 0원

### 실험 D: 설명 영상 — A와 병행

- **검증**: H1 (채널 확장)
- **방법**: 30~60초 화면 녹화. 기업 채용 사이트 폼이 자동으로 채워지는 목업 시나리오. 유튜브 Shorts + 틱톡 + 릴스 배포
- **성공 기준**: 랜딩 유입 전환율 >= 3%, 댓글/DM >= 20건
- **노력**: 1인 2일
- **기간**: 9일
- **비용**: 0원


## Go/No-Go 기준

| 시나리오 | 판단 |
|----------|------|
| A >= 40% AND B >= 25% | **Go** → MVP 개발 시작 |
| A >= 40% BUT B < 25% | 가격 모델 피벗 후 재실험 |
| A < 40% | 문제 정의 재검토 또는 **Kill** |
